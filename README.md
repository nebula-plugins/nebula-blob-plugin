nebula-s3-plugin
==============

Copy files to S3 using CopySpec

## Usage

### Applying the Plugin

To include, add the following to your build.gradle

    buildscript {
      repositories { jcenter() }

      dependencies {
        classpath 'com.netflix.nebula:nebula-s3-plugin:1.12.+'
      }
    }

    apply plugin: 'nebula-s3'
